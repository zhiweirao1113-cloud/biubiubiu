# 10.5 Gbps Visible Light Communication Systems Based on C-Plane Freestanding GaN Micro-LED

## 📝 摘要

本文展示了基于单个 c 平面独立 GaN 微 LED 的 10.5 Gbps 可见光通信系统。研究了从光电特性到通信性能的蓝色 GaN 衬底微 LED 的尺寸效应。较大尺寸的微 LED 可获得更高的光输出功率 (LOP)，但较小尺寸的微 LED 可实现更高的调制带宽。20 μm 微 LED 获得了约 2.3 GHz 的超高 -3 dB 带宽。通过改进 OFDM 信号带宽以有效利用 GaN 衬底微 LED 的高带宽，20、40、60、80、100 和 300 μm 微 LED 的最大数据速率分别为 8.294、8.394、10.547、10.019、9.398 和 7.294 Gbps。60 μm 微 LED 有望平衡 LOP 和带宽，实现 10.547 Gbps 的创纪录数据速率，这是据我们所知单个微 LED 获得的最高数据速率。基于 60 μm 微 LED 的最佳尺寸，还展示了 0.5、1、5 和 10 m 的扩展距离，数据速率分别为 10.045、9.668、9.277 和 8.649 Gbps。

## 🔬 核心贡献

1. **首次实现单个微 LED 超过 10 Gbps 数据速率** - 60 μm GaN 微 LED 实现 10.547 Gbps

4. **长距离传输验证** - 在 0.5-10 m 距离下验证系统性能

## 🔧 技术方法

### 器件制备
- **衬底**: 2 英寸 c 平面独立 GaN 衬底
- **外延结构**: Si 掺杂 n-GaN / 2 周期 InGaN/GaN MQW / AlGaN 电子阻挡层 / Mg 掺杂 p-GaN
- **尺寸**: 20, 40, 60, 80, 100, 300 μm
- **波长**: 450 nm (蓝光)

### 测试系统
```
发射端 (Tx):
  AWG (8 GS/s) → 电放大器 (38 dB, 4.2 GHz) → Bias-Tee → Micro-LED

接收端 (Rx):
  PIN 光电二极管 (1.4 GHz) → DSA (20 GS/s) → MATLAB 解调

调制方案:
  - OFDM + 预均衡 + 比特加载
  - BPSK 用于信道估计
  - QAM 用于数据传输
  - 循环前缀: 1/32 N_FFT
```

## 💡 核心发现

### 尺寸效应的物理机制

1. **小尺寸优势**:
   - 更均匀的电流扩展
   - 更好的散热
   - 更高的电流密度承受能力
   - 更短的载流子寿命 → 更高带宽

2. **大尺寸优势**:
   - 更高的光输出功率
   - 更好的信噪比潜力

3. **最优平衡点**:
   - 60 μm 在带宽 (1982 MHz) 和光功率 (4.57 mW) 之间达到最佳平衡
   - 频谱效率最高 (5.274 b/s/Hz)
