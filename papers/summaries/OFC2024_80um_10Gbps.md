## 📝 摘要

本文提出了一种基于 c 平面独立蓝色 GaN 衬底微 LED 的 VLC 系统。研究了 80 μm 微 LED 的光电特性和调制带宽性能，展示了其高调制带宽和高光输出功率特性。使用该微 LED 作为发射器，结合 OFDM 调制和比特加载算法，在 0.55 m 距离下实现了高达 10.009 Gbps 的数据速率。据我们所知，这是基于单像素微 LED 的 VLC 系统最高数据速率。

## 💡 技术亮点

### 1. 为什么选择 80 μm？

虽然 60 μm 实现了最高速率 (10.547 Gbps)，但 80 μm 仍有优势：

| 考虑因素 | 60 μm | 80 μm |
|---------|-------|-------|
| **数据速率** | 10.547 Gbps | 10.019 Gbps |
| **光功率** | 适中 | 更高 |
| **制造难度** | 较高 | 相对较低 |
| **可靠性** | 好 | 更好 |
| **应用场景** | 短距离高速 | 中长距离 |

### 2. 预均衡技术

```
频域预均衡 → 扩展信号带宽至 2 GHz
           → 提升高频部分能量
           → 改善 SNR 分布
```

### 3. 比特加载算法

```
子载波 SNR 估计 → 自适应比特分配 
               → 高 SNR 子载波：高阶 QAM
               → 低 SNR 子载波：低阶 QAM/BPSK
               → 频谱效率：5.0045 bit/s/Hz
```
